# Full Ecommerce using JS

## How to Run

1. Clone the repository and navigate to the `task10` directory:
   ```bash
   cd task10
   ```

2. Start a local server using any of the following methods:

   **Using Python:**
   ```bash
   python3 -m http.server 8000
   ```

   **Using Node.js (npx):**
   ```bash
   npx serve .
   ```

   **Using VS Code:**
   - Install the "Live Server" extension
   - Right-click `index.html` and select "Open with Live Server"

3. Open the URL shown in the terminal (e.g., `http://localhost:8000`) in your browser.

> **Note:** Opening `index.html` directly as a file will not work because the app uses ES modules (`import`/`export`), which require a server.
